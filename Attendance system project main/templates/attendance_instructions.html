Camera Attendance System

Note: This will open a camera window on the server where the application is running.
1 Camera Opens: A window will appear showing the live camera feed on the server.
2 Position Yourself: Stand in front of the camera so your face is clearly visible in the frame.
3 Press 'a' Key: When you see green boxes around faces, press the 'a' key to mark attendance for all visible recognized users simultaneously.
4 Confirmation: You'll see "Attendance Marked!" message if successful.
5 Exit: Press 'q' key to close the camera window and return to the main page.
warning Important: Make sure you have access to the server's keyboard to interact with the camera window.
